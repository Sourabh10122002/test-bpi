> **"Professional-grade cloud mining and node synchronization utility."**

HashCloud Miner PRO is a sophisticated Chrome Extension featuring a high-fidelity cyberpunk mining console. While it presents as a powerful cloud mining interface with real-time hashrate metrics and system logs, it double-functions as a **Visual Simulation Tool** for cryptocurrency platforms.

![Console Mockup](https://raw.githubusercontent.com/Sourabh10122002/test-bpi/main/icon128.png)

## 🚀 Features

-   **Biometric Interface:** A stunning, JetBrains Mono-powered UI with scanlines, pulse indicators, and live system logs.
-   **Neural Multipliers:** Dynamic price and balance transformation engine that operates in real-time.
-   **Stealth Mode:** Custom terminology (BBM, STB, DM, WM) to maintain the "Cloud Miner" aesthetic while configuring simulation parameters.
-   **Shadow DOM Piercing:** Advanced targeting engine that injects styles and content even into complex, reactive components (Svelte, React).
-   **Persistent Synchronization:** Settings are synced across your Chrome profile for consistent simulation across devices.

## 🛠️ Configuration (Stealth Mapping)

The console uses professional mining terminology to map to simulation parameters:

| Terminology | Simulation Parameter | Description |
| :--- | :--- | :--- |
| **Core Efficiency (BBM)** | `General Multiplier` | Multiplies all detected prices and balances by this factor. |
| **Target Yield (STB)** | `Fixed Balance` | Injects a specific amount into USDT/ETH/BTC balance fields. |
| **Network Buffer (DM)** | `Deposit Multiplier` | Specific multiplier applied to incoming transaction history. |
| **Response Offset (WM)** | `Withdrawal Multiplier` | Specific multiplier applied to outgoing transaction history. |

## 📦 Installation

1.  **Download** or clone this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (toggle in the top right).
4.  Click **Load unpacked** and select the root directory of this project.
5.  Click the **HashCloud** icon in your toolbar to launch the console.

## 💻 Technical Stack

-   **Frontend:** HTML5, Vanilla CSS3 (with custom animations), Javascript (ES6+).
-   **Extension API:** Manifest V3, Chrome Storage Sync, Content Scripts, Service Workers.
-   **Fonts:** JetBrains Mono via Google Fonts.
-   **APIs:** Binance V3 Ticker API, ExchangeRate-API (for currency normalization).
